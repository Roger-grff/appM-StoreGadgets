# 📱 RG Gadgets Store

Aplicación móvil para gestión de una tienda de gadgets, desarrollada con **Ionic + Angular + Capacitor** y base de datos en **Supabase**.

## 🚀 Creación del proyecto desde cero

### 1. Instalar Ionic CLI globalmente

```bash
npm install -g @ionic/cli
```

### 2. Crear el proyecto

```bash
ionic start appM-StoreGadgets blank --type=angular --capacitor
```

> Seleccionar `standalone components` cuando lo pregunte.

### 3. Entrar al proyecto

```bash
cd appM-StoreGadgets
```

---

## 📦 Instalación de dependencias

### Supabase (base de datos y storage)

```bash
npm install @supabase/supabase-js
```

### Capacitor Splash Screen

```bash
npm install @capacitor/splash-screen
```

### Capacitor Assets (generación de íconos y splash)

```bash
npm install @capacitor/assets --save-dev
```

### Sincronizar Capacitor con Android

```bash
npx cap sync android
```

## 📄 Generación de páginas

Las páginas se generaron con el CLI de Ionic:

```bash
# Página de listado de gadgets
ionic generate page pages/gadgets

# Página de formulario (crear/editar)
ionic generate page pages/gadgets-form

# Página de detalle de gadget
ionic generate page pages/gadget-detalle
```

### Generación del servicio

```bash
ionic generate service services/supabase
```

## ⚙️ Configuración de Supabase

### 1. Crear proyecto en Supabase

Ir a [supabase.com](https://supabase.com/), crear un proyecto y copiar la **URL** y la **anon key**.

### 2. Configurar variables de entorno

En `src/environments/environment.ts`:

```typescript
export const environment = {
  production: false,
  supabaseUrl: 'TU_SUPABASE_URL',
  supabaseKey: 'TU_SUPABASE_ANON_KEY'
};
```

### 3. Crear la tabla `gadgets` en Supabase

Ejecutar en el **SQL Editor** de Supabase:

```sql
create table gadgets (
  id bigint generated by default as identity primary key,
  nombre text not null,
  marca text not null,
  modelo text,
  precio numeric not null,
  stock integer not null,
  categoria text,
  descripcion text,
  imagen_url text,
  video_url text,
  audio_url text
);
```

### 4. Crear los buckets de Storage en Supabase

Crear estos dos buckets como **públicos**:

- `gadgets-imagenes` → para imágenes de los gadgets
- `gadgets-audios` → para audios de la app

## 🧭 Rutas de la aplicación

| Ruta | Componente | Descripción |
|---|---|---|
| `/gadgets` | `GadgetsPage` | Listado principal con buscador |
| `/gadgets-form` | `GadgetsFormPage` | Formulario para crear gadget |
| `/gadgets-form/:id` | `GadgetsFormPage` | Formulario para editar gadget |
| `/gadget-detalle/:id` | `GadgetDetallePage` | Detalle completo del gadget |

## 📱 Compilar para Android

### 1. Compilar el proyecto web

```bash
ionic build
```

### 2. Sincronizar con Android

```bash
npx cap sync android
```

### 3. Abrir en Android Studio

```bash
npx cap open android
```

### 4. Ejecutar en dispositivo o emulador

Desde Android Studio presionar el botón **Run ▶** o usar:

```bash
npx cap run android
```

## 🖼️ Generar ícono y splash screen

Coloca tus imágenes en la carpeta `resources/`:

- `resources/icon.png` → mínimo **1024x1024 px**
- `resources/splash.png` → mínimo **2732x2732 px**

Luego ejecutar:

```bash
npx capacitor-assets generate --android
npx cap sync android
```
